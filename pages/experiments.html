🤖 Telegram Bot + Jira (MVP)

Разработан Telegram-бот с полноценной интеграцией Jira API. Проект доведён до уровня MVP и полностью функционален — не запущен в production по внутренним причинам компании.

Реализовано:

  • Подключение к Jira API — поиск задач, получение статусов, данных по заявкам.
  • Двухэтапная аутентификация (/start) — проверка корпоративной почты и пароля через базу данных, доступ только авторизованным пользователям.
  • Быстрый поиск заявок:
    • /where — поиск в проекте BO (ИНН, статус, номер заявки, юр. название, автор, исполнитель).
    • /where_buh — поиск в проекте GTE для бухгалтерии.
    • /where_lk — выдача логина и пароля от личного кабинета партнёра.
  • Автоматическая рассылка Excel-отчётов прямо в Telegram (ежедневные, еженедельные, ежемесячные).
  • Интеграция с FastAPI DBC System для хранения данных пользователей и статистики.
  • Сбор статистики по заявкам (по ИНН, по проектам, по времени обработки).

Статус: MVP готов. Не запущен в production.

⚡ FastAPI Platform (MVP)

Разработана backend-платформа как единый центр интеграций между Jira и Telegram Bot. Проект доведён до уровня MVP.

Реализовано:

  • Backend на FastAPI для интеграции с Jira и Telegram Bot.
  • REST API для обработки запросов и управления заявками.
  • Интеграция с DBC System (Data Base Control System) — хранение данных пользователей и логов обращений.
  • Централизованная обработка команд Telegram Bot.
  • Сбор статистики по заявкам и активности пользователей.

Статус: MVP готов. Не запущен в production.
